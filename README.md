# dotfiles for hexagonforce 

## Requirements
- zsh
- GNU stow
- Python 3

## Applications configured
- conky
- fcitx5 (with fcitx-hangul)
- firefox
- foot
- git
- lf (chafa, sixel)
- mpd
- mpv
- ncmpcpp
- nsxiv
- nvim
- sway
- waybar
- wofi
- ytfzf (ueberzugpp)
- zsh

## Usage
Set the default shell to zsh, then run `./dotfiles.sh`.
